# OOS Watch

Taobao补货提醒（Chrome扩展程序）

## Generate the .crx

```shell
npm i
npm run build
```

## Develop

```shell
# start for the static contents
npm start

# start for a chrome extension
# then "load unpacked" on chrome(./dist/main)
npm run build
npm run dev
```
